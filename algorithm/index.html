# 写在前面的话

很多人觉得算法可能对于前端工程师要求并不是很高，但是作为一名程序员，算法就像是一个人的内功，一座大厦的地基，只有内功足够深厚，地基足够坚固，才能写出优秀高效的代码，同时这也是一名程序员必备的技能，包括现在的许多大厂，国外的google，facebook，国内的字节跳动，算法是作为首先考察的内容，因为它能很快的看出一个人的“功底”，从而快速地筛选掉一大批候选人，从而节约面试效率。所以说这些公司薪酬高是 “情有可原” 的... 况且不说这种面试模式的好坏，我们作为候选人只能去适应这种游戏规则，才能 “混口饭吃” 。所以要想让自己不被迅速发展互联网大环境给淘汰，平时多刷刷算法题吧... 总结起来就是一句话：

不爱刷也得刷，硬着头皮也要刷！
