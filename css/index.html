# 骨灰级方法

在父元素里最后添加一个<div style="clear:both"><div>来解决，但这样我们的HTML代码里就多了些冗余的代码且从语义化角度来看这也是不合理的。

# clearfix方法

我们可以通过创建一个.clearfix的类来完美解决问题：

.clearfix:after{
    content:'.';
    display:block;
    height:0;
    visibility:hidden;
    clear:both;
}

解释如下：

  1. display:block使生成的元素以块级元素显示,占满剩余空间;

  2. height:0 避免生成内容破坏原有布局的高度。

  3. visibility:hidden 使生成的内容不可见，并允许可能被生成内容盖住的内容可以进行点击和交互;

4）通过 content:"."生成内容作为最后一个元素，至于content里面是点还是其他都是可以的，例如oocss里面就有经典的 content:"XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX",有些版本可能content 里面内容为空,一丝冰凉是不推荐这样做的,firefox直到7.0 content:”" 仍然会产生额外的空隙。

# CSS瀑布流效果

首先查看效果:

# figure标签

<figure> 标签规定独立的流内容（图像、图表、照片、代码等等）

<figcaption> 规定下方说明文字

figure 元素的内容应该与主内容相关，但如果被删除，则不应对文档流产生影响。

# CSS3多列布局

column-width：规定每列宽度

column-count：规定元素应该被分割的列数，默认为3

column-gap：规定列之间的间距

# HTML结构

<div id="columns">
  <figure>
    <img src="cinderella.jpg" alt>
    <figcaption>Cinderella wearing European fashion of the mid-1860’s</figcaption>
  </figure>
  <figure>
    <img src="rapunzel.jpg" alt>
    <figcaption>Rapunzel, clothed in 1820’s period fashion</figcaption>
  </figure>
  …
</div>

# CSS样式

给每一列设置固定的宽度能使这个瀑布流布局的响应式效果更好。浏览器在每次渲染时都会计算一行能够放多少列320像素的单列。

#columns {
	column-width: 320px;
	column-gap: 15px;
	width: 90%;
	max-width: 1100px;
	margin: 130px auto;
}

# 对移动设备的支持

最后，我们通过@media query来设置对小屏幕设备的支持：

@media screen and (max-width: 750px) { 
	#columns { column-gap: 0px; }
	#columns figure { width: 100%; }
}

# 用CSS绘制三角形

# 原理

假设有一个宽高为定值的矩形，通过分别设置border，不同颜色，会得到这样的效果：

HTML代码如下：

<div class="triangle"></div>
1

CSS代码如下：

.triangle{
  width:100px;
  height:100px;
  border-style:solid;
  border-width:20px;
  border-color:red blue yellow black;
}

再将左右的border-color设为transparent，去掉border-top，就变成一个向上的三角形了，想要哪边就画哪边的border,并且让它相邻两边的border-color:transparent 。

CSS代码如下：

.triangle{
  width:0px;
  height:0px;
  border-style:solid;
  border-width:0 20px 20px;
  border-color:transparent transparent blue;
}

# 画提示框三角形

# 原理

先画一个三角形，再画白色三角形的，调整几像素位置，覆盖掉一边

CSS代码如下：

.triangle{
  position:relative;
  width:100px;
  height:50px;
  border:2px solid black;
  border-radius:10px;
}

.triangle:before{
  position:absolute;
  content:'';
  top:-10px;
  left:25px;
  border-style:solid;
  border-width:0 10px 10px;
  border-color:transparent transparent black;
}

.triangle:after{
  position:absolute;
  content:'';
  top:-8px;
  left:25px;
  border-style:solid;
  border-width:0 10px 10px;
  border-color:transparent transparent white; 
}
