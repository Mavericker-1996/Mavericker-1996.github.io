# #39 不用循环生成数组

# 题目描述

完成 arrWithoutLoop 函数，它会被传入一个整数 n 作为参数，返回一个长度为 n 的数组，数组中每个元素的值等于它的下标。arrWithoutLoop 中不能使用循环控制结构。

（本题来源：UC 国际前端笔试题）

const arrWithoutLoop = (n) => /* TODO */
console.log(arrWithoutLoop(5)) //[0,1,2,3,4]
1
2

# 解题思路

由于不让用循环，首先想到的自然是递归了（在不考虑使用ES6中map函数的情况下）

# 代码

第一版就是在函数里面声明一个递归函数，然后调用，代码比较麻烦：

const arrWithoutLoop = (n) => /* TODO */ 
{
  var res=[]; //结果数组
  if(n===0) return res; //如果n等于0，直接返回空数组
  const fn = (n) => { //使用递归函数来代替循环
  	// console.log(n);
  	if(n>0) res.unshift(n-1); 
  	else return res;
  	fn(n-1);
  }
  fn(n);
  return res;;
}

const arrWithoutLoop = (n) => n===0 ? [] : arrWithoutLoop(n-1).concat(n-1);

const arrWithoutLoop = (n) => [...Array(n)].map((v,i)=>i) //Array(n)会直接返回一个长度为n的空数组，所以我们要使用...将其转化成一个数组。

例如，输入 emoji.png，返回 .png。输入.emoji或者emoji就不返回。

const extname = (filename) => {
  /* TODO */
}

# 解题思路

直接用es6中的lastIndexOf查找最后一个.，判断下标是否大于0，如果是的话，就将后面字段输出即可。

# 代码

const extname = (filename) => {
  /* TODO */
  return filename.lastIndexOf('.')>0 ? filename.slice(filename.lastIndexOf('.')) : ''
}

# 解题思路

这题考察的是es6中标签模板的知识，具体内容可以参考这里，举个里面的例子：

let a = 5;
let b = 10;

tag`Hello ${ a + b } world ${ a * b }`;
// 等同于
tag(['Hello ', ' world ', ''], 15, 50);

所以我们就可以在模板字符串中的内容外面加上span标签和对应样式进行操作即可。

# 代码

const highlight = (literals,...args) =>{
	console.log(literals);
	var res = '';
	for(var i=0;i<args.length;i++){
		res += literals[i]+`<span style="color:red">${args[i]}</span>`
	}
	res += literals[i]; //literals的长度永远比args的长度多一
	console.log(res);
	return res;
}

# #24 +1s程序

# 题目描述

完成一个生成计数器的函数 plusFor，调用它会返回一个计数器。计数器本身也是一个函数，每次调用会返回一个字符串。

达到以下的效果：

const counter1 = plusFor('小明')
counter1() // => 为小明+1s
counter1() // => 为小明+2s
counter1() // => 为小明+3s
...

const counter2 = plusFor('李梅')
counter2() // => 为李梅+1s
counter2() // => 为李梅+2s
counter2() // => 为李梅+3s
...

# 解题思路

# 代码

const plusFor = (name) => {
	var second = 0;
	return () => `为${name}+${++second}s`
}
